# Workspace Rules for Clarior

## Git Remote Configuration
Whenever pushing code or committing changes:
- Always push to **both** remotes:
  1. `git push origin <branch>` (https://github.com/shuganchauhan/Clarior.git)
  2. `git push upstream <branch>` (https://github.com/Ashwin-Agrawal/Clarior.git)
